## 題目

給你一個字串代表每格的顏色、以及一個陣列代表把該格拿掉的代價

求使每格周圍兩格都跟自己顏色不一樣的最小代價

> n <= 10<sup>5</sup>

## 詳解

翻譯一下題目

> 求使每格周圍兩格都跟自己顏色不一樣的最小代價

相當於
>對於連續 k 個相同顏色的格子，我們需要把代價最高的留下來，其他都拿掉

所以不斷紀錄某個連續段的代價總和以及最大代價

然後把代價總和減掉最大代價

就會是這段的最小代價

不斷處理直到全部都看過一遍

恭喜AC :D

## 程式碼

```python
class Solution:
    def minCost(self, colors: str, neededTime: List[int]) -> int:
        n = len(colors)
        curr_color = colors[0]
        total_sum = 0
        single_color_sum = 0
        single_color_max = 0
        for i in range(n):
            if colors[i] != curr_color:
                total_sum += single_color_sum - single_color_max
                curr_color = colors[i]
                single_color_sum = 0
                single_color_max = 0
                

            single_color_sum += neededTime[i]
            single_color_max = max(single_color_max, neededTime[i])

        total_sum += single_color_sum - single_color_max
        return total_sum 
```
